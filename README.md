# BLACKPINK AREA — 2026 Fan Edition

A modern BLACKPINK-inspired fan website redesign created by **Rami Almohamad**.

## Live Project

After uploading the files to GitHub Pages, the site will be available at:

https://ramialmoh12.github.io/blackpink-area/

## Overview

This project is a full redesign of the older BLACKPINK Area website. The new version uses a darker 2026 visual style with neon pink details, cinematic sections, smooth JavaScript animations, responsive layout, and a mobile burger menu.

The goal is to make the website look more professional, impressive, and suitable for a front-end developer portfolio.

## Features

- Modern BLACKPINK-inspired black and pink theme
- 2026 dark luxury fan concept
- Sticky glass navigation bar

- Smooth scroll behavior

## How to Upload to GitHub Pages

1. Open your `blackpink-area` repository on GitHub.
2. Replace the old files with the new files:
   - `index.html`
   - `style.css`
   - `app.js`
   - `README.md`
3. Commit the changes.
4. Wait 1–3 minutes.
5. Open your GitHub Pages link and refresh the page.
